# Hitko Theme

A minimal theme for VSCode.

![preview](preview.png)

</div>

## Installation

1. Install theme from the [Marketplace](https://marketplaPce.visualstudio.com/items?itemName=Hitko.hitko-theme)
2. Go to `File > Preferences > Color Theme`
3. Select `Hitko Theme`
